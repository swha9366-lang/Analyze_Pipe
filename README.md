# Analyze_Pipe — B31.1 Piping Stress Analyzer (v24)

브라우저에서 동작하는 ASME B31.1 배관응력해석 프로그램입니다. 설치 없이 아래 주소로 접속해 바로 사용할 수 있습니다.

**▶ 실행:** https://swha9366-lang.github.io/Analyze_Pipe/

## 주요 기능
- 3D 보요소 유한요소 해석 (절점당 6자유도, 직접강성법)
- ASME B31.1 코드체크: Eq.15 지속하중 · Eq.14 열팽창 응력범위 · Eq.16 일시하중(지진 0.3g)
- 압력설계 두께 검증(tm), 지지 간격 검토(Table 121.5), 행어 자동 사이징(MSS SP-58 25% 규칙)
- 지지장치: 앵커(3축 힘/모멘트/변위 입력 가능) · 레스트 · 가이드(갭 mm, 접촉해석) ·
  라인스톱 · 스프링/콘스턴트 행어 · 스너버(방향 X/Y/Z, 정격하중 검증 W331)
- 부속: 엘보/벤드(SIF·유연계수) · 티 · 리듀서 · 밸브(무게 입력 가능) · 플랜지 · 집중질량/하중
- 엔진 보정 옵션 D1~D7 (v21 호환 모드 지원)
- 3D 시각화(응력비 컨투어·변형 애니메이션), 자동 보고서(Word), 개선 어드바이저
- AutoPIPE 연동: NTL/PCF 내보내기 + NTL/DAT 불러오기 (드래그&드롭 지원)

## 사용법 (빠른 시작)
1. 좌측에서 재질·관경·설계조건(P, T) 입력
2. 절점 좌표 입력으로 배관 경로 모델링 (직관/밴드/분기)
3. 절점에 부속·지지장치 추가 (칩 클릭으로 수정 가능)
4. **[해석 실행]** → 응력비·반력·변위 확인, 보고서 출력
5. 모델은 `.json`으로 저장/불러오기 (AutoPIPE `.ntl`/`.dat`도 불러오기 가능)

샘플 모델(A~D)이 내장되어 있어 바로 체험할 수 있습니다.
